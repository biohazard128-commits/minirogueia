Mini Rogue — Préparation Dataset IA

Cette page sert à préparer un dataset de détection d'objet pour entraîner un modèle TFLite capable de détecter une carte Mini Rogue.

Étape 1 — Capturer les photos

Photos capturées : 0 / 50

Checklist de variations obligatoires

Étape 2 — Annotation

  1. Ouvre Edge Impulse ou Roboflow
  2. Crée un projet Object Detection
  3. Importe les photos
  4. Dessine un rectangle autour de chaque carte
  5. Utilise toujours le label : carte
Le rectangle doit entourer uniquement la carte, sans trop de fond autour.

Étape 3 — Entraînement

  1. Lancer l'entraînement
  2. Sélectionner le modèle Fast
  3. Attendre la fin de l'entraînement
  4. Exporter le modèle en TFLite
Le modèle Fast est suffisant pour détecter un rectangle de carte sur mobile.

Plateformes recommandées

  • Edge Impulse
  • Roboflow
